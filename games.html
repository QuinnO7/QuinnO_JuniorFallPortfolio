My Games

This page showcases a collection of the games that I have made throughout this process completed in Web & App Development. Each game highlights different skills in coding, creativity, and problem-solving. These games were difficult but also fun to make. Also seeing that I was able to have working and playable was really nice because I couldn't imagine myself being able to do that.

Space Blaster

This game is my favorite game and also favorite thing that I have worked on because it demonstrates creativity and my skills to be able to make a game. I created a space themed game where you are able to control this spaceship and you have to defeat all of the enemies to progress to the next level. There are 4 levels and a boss fight at the end. This was a difficult but fun process and it ended up paying off.

Skills: Unity, Game Design

Forest Adventure

In this game, you have to go around the forest getting all of the collectibles and making it to the end to win the game. I didn't really get to expand on my idea for this game because I was running out of time so I did the best that I could do with what I had. This helped me improve my understanding of time mangement and going in depth about game design.

Skills: Unity, Game Design
