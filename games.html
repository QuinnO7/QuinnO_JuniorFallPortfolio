My Games

This page showcases games I developed during my Web & App Development coursework. These projects demonstrate skills in programming, game design, problem-solving, and creativity using Unity.

Space Blaster

Space Blaster is a space-themed action game where the player controls a spaceship and must defeat enemies to progress through multiple levels. The game includes four levels and a final boss fight.

This project helped me develop stronger skills in game mechanics, level design, and problem-solving under development constraints.

Skills: Unity, Game Design, Level Design

Forest Adventure

Forest Adventure is a collectible-based exploration game set in a forest environment. The objective is to collect items and reach the end of the level to complete the game.

Due to time limitations, the project scope was reduced; however, it provided valuable experience in planning, prioritization, and time management in game development.

Skills: Unity, Game Design, Time Management
